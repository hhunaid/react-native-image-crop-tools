## Instructions

Clone the project

```bash
git clone https://github.com/hhunaid/react-native-image-crop-tools.git
```

Enter example folder
```bash
cd example
```

Install dependencies
```bash
yarn install

or

npm install
```

Install native modules
```bash
cd ios 
pod install
```

Run Metro
```bash
npm run start
```

Run on iOS or Android simulator
```bash
npm run ios 

or 

npm run android
```
